# Graceful Handoff Help

## What is Graceful Handoff?

Creates detailed conversation summaries for seamless model handoffs. When you need to switch sessions or models, `/grace` generates comprehensive context so work can continue without losing progress.

## Usage

```
/grace
```

## What It Produces

1. **Task list** - Summarizes all work in the conversation
2. **Code & architecture notes** - Technical details for each task
3. **Action plan** - Recovery steps for the next session

## When to Use

- Before context window fills up
- When switching between models
- To checkpoint complex work
- Before ending a long session
