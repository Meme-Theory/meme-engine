# ==============================================================================
#  Requirements: MCP-server environment (system Python 3.13)
# ==============================================================================
#
# This file covers every dependency needed to run the MCP servers configured in
# the project's .mcp.json. The servers are stdio processes spawned by Claude
# Code; they do NOT need GPU, torch, or anything in the numerical compute stack.
#
# ---- INSTALL ----------------------------------------------------------------
#
#   # Windows:
#   py -3.13 -m pip install -r requirements-mcp.txt
#
#   # macOS / Linux:
#   python3.13 -m pip install -r requirements-mcp.txt
#
#   # paper-search-mcp is a local editable package (installed by the MCP unfold
#   # when the paper-search MCP is selected); install separately if present:
#   py -3.13 -m pip install -e tools/mcp-servers/paper-search-mcp
#
# ---- WHERE USED -------------------------------------------------------------
#
#   tools/mcp-servers/*/server.py   - the MCP servers configured in .mcp.json
#
# ---- NOT FOR ----------------------------------------------------------------
#
#   Numerical compute scripts - use requirements-compute.txt installed into the
#   project's compute venv (if a compute directory was scaffolded).
#
# ---- PIN RATIONALE ----------------------------------------------------------
#
#   Versions pin to a tested Windows 11 / Python 3.13 set. The MCP SDK (`mcp`)
#   and `fastmcp` APIs have churned across minor releases - loosening these pins
#   is a gamble. If you bump, verify with `/mcp` that every server still
#   connects and at least one tool-call round-trip succeeds.
#
# ---- DISCIPLINE ADDITIONS ---------------------------------------------------
#
#   The selected discipline pack may append lines below. Each appended line
#   includes a `# Used by:` comment pointing to the MCP server that needs it.
#   Extras you don't use are harmless (tens of MB on disk).
#
# ==============================================================================
